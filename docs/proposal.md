# 租房安心审创意提案

## 1. 项目名称与口号

项目名称：租房安心审

口号：签租房合同前，先让 AI 帮你看一遍。

定位：面向毕业生、年轻租客、老人和普通家庭的 AI 租房合同解读器。

赛道：社会服务赛道。

## 2. 解决的问题

租房合同里的风险常常藏在押金、涨租、维修、提前解约、滞纳金、管辖地和格式条款中。很多租客第一次租房时没有经验，签字后才发现押金难退、房东可随意涨租、维修责任全归自己、房东可不经同意进屋。

租房安心审解决三个关键问题：

- 看不懂：把复杂条款翻译成大白话，说明对租客意味着什么。
- 找不全：自动识别押金扣款、涨租陷阱、维修转嫁、换锁收房等租房高频坑点。
- 不会谈：给出可复制的替代条款和谈判话术，让租客知道怎么改、怎么说。

## 3. 核心功能

- 租房合同输入：支持粘贴合同正文，后续可扩展 PDF、Word、图片 OCR。
- 押金陷阱识别：识别退还周期过长、固定保洁费、折旧补偿、单方扣款等问题。
- 涨租预警：识别自动续租涨价、租期内单方调价、不同意就搬离等风险。
- 维修责任分析：区分自然损耗和人为损坏，避免房屋老化维修全转嫁给租客。
- 居住权保护：识别房东随意入户、换锁收房、带人看房不经同意等风险。
- 大白话解读：每条风险都说明“这句话会让你吃什么亏”。
- 谈判话术：告诉用户哪些条款可以谈、怎么谈、底线在哪。
- 一键采纳修改：把风险条款替换成更合理的租客保护条款。
- 报告导出：生成租房合同解读报告，方便沟通、留痕和维权。

## 4. 技术方案

前端使用 React 构建租房合同解读工作台，包含合同输入、风险高亮、评分面板、知识库、风险卡片、采纳修改和报告导出。

当前 Demo 采用“规则库 + AI + 原文证据校验”的结构：

1. 输入租房合同。
2. 自动识别合同类型，默认使用房屋租赁合同知识库。
3. 使用租房专属规则命中押金、涨租、维修、解除、管辖等风险。
4. 对每个风险绑定合同原文证据，避免凭空编造。
5. 可选调用大模型生成解释、替代条款和谈判话术。
6. 对 AI 返回结果做原文证据校验，证据不存在的风险会被过滤。
7. 输出风险评分、风险清单、修改建议和可导出的解读报告。

后续真实产品可补充：

- OCR 服务：支持手机拍照识别租房合同。
- 城市租金数据：提供小区和周边租金参考。
- 押金计算器：根据退租情况估算应退押金。
- 维权入口：提供住建部门、12345、调解和法院路径。
- 房东信用档案：沉淀匿名租客评价和纠纷记录。

## 5. 预期效果

- 帮助租客在签字前识别主要合同陷阱。
- 降低押金纠纷、涨租纠纷和维修推诿带来的损失。
- 让毕业生、老人等弱势租房群体更容易看懂合同。
- 把“发现问题”变成“知道怎么谈、怎么改”。
- 形成标准化解读报告，方便租客与房东、中介沟通。

## 6. 创新亮点

- 聚焦租客视角，而不是面向企业法务的通用合同审查。
- 使用租房专属规则库，覆盖押金、涨租、维修、入户、换锁、管辖等真实痛点。
- 每条风险都必须绑定合同原文证据，降低 AI 幻觉。
- 输出大白话解释和谈判话术，降低普通人使用门槛。
- 可扩展到押金计算器、房东信用档案、租金行情对比和维权通道。
- 兼具社会服务价值和清晰商业落地路径。
