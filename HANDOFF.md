# 租小审项目交接说明

## 项目概况

项目名称：租小审

定位：AI 租房避坑助手 + 退租证据包助手

赛道：社会服务赛道

当前形态：React + Vite 单页 Demo，主打租客视角的租房合同避坑和退租证据整理工作台。

## 技术栈

- React 19
- Vite
- lucide-react
- 原生 CSS

## 核心文件

- `src/App.jsx`：主界面、租房合同审查逻辑、风险建议、采纳修改、AI 接口配置、退租证据包。
- `src/App.css`：浅色专业产品 UI、按钮、风险卡片、AI 配置面板、响应式布局。
- `src/index.css`：全局基础样式。
- `tests/app.spec.js`：Playwright 端到端测试，覆盖审查压测、入住验房和补贴匹配。
- `playwright.config.js`：端到端测试配置。
- `docs/proposal.md`：参赛创意提案。
- `README.md`：项目说明。
- `package.json`：依赖和运行命令。

## 当前功能

1. 租房合同文本输入和示例合同一键载入。
2. 默认房屋租赁合同、默认租客视角、默认严格风控。
3. 15+ 条租房专属风险规则 + 智能检查，覆盖押金扣款、涨租陷阱、维修责任、提前解除、入户、管辖、格式条款等。
4. 风险评分和维度分析。
5. 合同原文风险关键词高亮。
6. 每条风险有原文证据、大白话解释、替代条款和谈判话术。
7. 每条建议可点击“采纳并改写合同”。
8. 支持“全部采纳”一键应用所有修改建议，并生成可导出的修订版合同草案。
9. 支持 TXT / MD / PDF / DOCX 合同文件导入，PDF 和 DOCX 在浏览器本地解析文本。
10. 默认 Mock AI 演示模式，评委无需 API Key 也能看到 AI 审查链路。
11. “开始演示”会直接载入示例合同、生成 Mock AI 审查结果、展开看房清单并运行样本压测。
12. AI 大模型接口配置入口，包括服务商、Base URL、模型名、API Key。
13. 真实 API 模式支持 5 份代表样本批量压测，记录 JSON 解析、证据命中、幻觉过滤、合同类型误判和耗时。
14. AI 结果会做原文证据校验和质量自检，过滤证据不存在的幻觉风险。
15. 12 份内置租房样本压测，覆盖整租、合租、公寓托管、中介格式、二房东、续租、退租、杂费、维修、免责和低风险合同。
16. 一键生成正式 HTML 报告，可打印或另存 PDF，合并审查结果、采纳建议、看房清单、入住验房、退租证据、押金估算、补贴线索和样本压测。
17. 退租证据包 Tab：退租信息录入、证据清单勾选、完整度统计、缺口提示、自动保存、证据包摘要复制和导出。
18. 退租沟通生成器：押金退还、维修争议、退租交接三种场景，支持生成后复制。
19. 下一步行动清单，支持逐项完成状态切换。
20. 创意提案页面：市场痛点、技术方案、竞品对比、商业化路径、产品路线图和可输入押金计算器。
21. 看房检查清单：覆盖入户、水电燃气、墙面地板、家具家电、厨房卫生间和周边环境，支持折叠展开和进度统计。
22. 入住验房 Tab：支持房屋类型选择、6 个房间 × 7 项检查、照片上传预览、完好/瑕疵/跳过状态、瑕疵类型和备注。
23. 验房报告生成：自动汇总瑕疵，生成房东确认话术，支持复制和 txt 导出。
24. 退租证据包联动入住验房：照片分组提示可使用入住验房报告作为证据来源。
25. 毕业生租房补贴匹配：覆盖 12 个城市，支持城市选择、个人情况输入、政策线索排序、申请材料、来源和更新时间提示。
26. 长合同性能优化：证据高亮限制预览长度和命中数量，输入审查计算使用延后值，“全部采纳”改为单次合同更新，降低主线程卡死风险。

## 可检查重点

1. UI 是否显示“租小审”。
2. 默认示例是否为租房合同，而不是服务合同。
3. 默认审查画像是否为房屋租赁合同、租客方、严格风控。
4. 押金、涨租、维修、入户、换锁、管辖、解释权等风险是否能命中。
5. 风险证据是否都来自合同原文。
6. “采纳并改写合同”和“全部采纳”的交互是否合理。
7. 退租证据包四个子模块是否能切换、勾选、生成沟通说明并导出摘要。
8. 入住验房 Tab 是否能切换房间、标记瑕疵、生成报告、复制话术和导出报告。
9. 看房检查清单是否能展开、勾选并实时更新进度。
10. 创意提案中的补贴匹配工具是否能输入情况并排序展示政策线索。
11. 一键演示是否直接生成 Mock AI 结果、质量自检和样本压测。
12. TXT 文件导入是否能把合同正文填入审查区。
13. 长合同导入、审查和全部采纳是否仍保持响应。
14. 移动端响应式是否正常。
15. 如果用于 AI 创造力大赛，还可以补真实 OCR、维权通道、聊天记录导入等展示亮点。
