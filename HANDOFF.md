# 租房安心审项目交接说明

## 项目概况

项目名称：租房安心审

定位：AI 租房合同解读器

赛道：社会服务赛道

当前形态：React + Vite 单页 Demo，主打租客视角的租房合同避坑工作台。

## 技术栈

- React 19
- Vite
- lucide-react
- 原生 CSS

## 核心文件

- `src/App.jsx`：主界面、租房合同审查逻辑、风险建议、采纳修改、AI 接口配置。
- `src/App.css`：浅色专业产品 UI、按钮、风险卡片、AI 配置面板、响应式布局。
- `src/index.css`：全局基础样式。
- `docs/proposal.md`：参赛创意提案。
- `README.md`：项目说明。
- `package.json`：依赖和运行命令。

## 当前功能

1. 租房合同文本输入和示例合同一键载入。
2. 默认房屋租赁合同、默认租客视角、默认严格风控。
3. 15+ 条租房专属风险规则 + 智能检查，覆盖押金扣款、涨租陷阱、维修责任、提前解除、入户、管辖、格式条款等。
4. 风险评分和维度分析。
5. 合同原文风险关键词高亮。
6. 每条风险有原文证据、大白话解释、替代条款和谈判话术。
7. 每条建议可点击“采纳并改写合同”。
8. 支持“全部采纳”一键应用所有修改建议。
9. AI 大模型接口配置入口，包括服务商、Base URL、模型名、API Key。
10. AI 结果会做原文证据校验，过滤证据不存在的幻觉风险。
11. 创意提案页面和租房产品路线图。

## 运行方式

```bash
npm install
npm run dev
```

默认预览地址：

```text
http://localhost:5173/
```

## 验证建议

```bash
npm run build
npm run lint
```

## 可检查重点

1. UI 是否显示“租房安心审”。
2. 默认示例是否为租房合同，而不是服务合同。
3. 默认审查画像是否为房屋租赁合同、租客方、严格风控。
4. 押金、涨租、维修、入户、换锁、管辖、解释权等风险是否能命中。
5. 风险证据是否都来自合同原文。
6. “采纳并改写合同”和“全部采纳”的交互是否合理。
7. 移动端响应式是否正常。
8. 如果用于 AI 创造力大赛，还可以补押金计算器、拍照识别、维权通道等展示亮点。
