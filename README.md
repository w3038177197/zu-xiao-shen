# 租房安心审

租房安心审是一个面向社会服务赛道的 AI 租房合同解读器 Demo。它把租房合同输入、押金风险识别、涨租预警、维修责任分析、谈判话术和参赛提案集中在一个可演示的 React 工作台里。

## 运行

```bash
npm install
npm run dev
```

默认访问 Vite 输出的本地地址。

## 当前 Demo 已实现

- 租房合同粘贴与示例合同一键载入
- 房屋租赁合同自动识别和租客视角严格审查
- 15+ 条租房专属风险规则命中，覆盖押金、涨租、维修、解除、入户、管辖等
- 风险评分与维度分析，包括押金风险、涨租风险、解除风险、维修风险等
- 合同原文风险关键词高亮，一眼看出哪里有问题
- 逐条风险解释、证据片段、替代条款和谈判话术
- 一键采纳修改建议，生成修订版合同草案并支持导出
- AI 大模型接口配置，可接入 OpenAI-compatible 网关
- AI 证据原文校验和质量自检，避免输出原文不存在的幻觉风险
- 可输入押金计算器，按未结费用、扣款凭证和正常损耗估算应退押金
- 创意提案页与产品路线图展示
