# 租小审

租小审是一个面向普通租客的 AI 租房全流程风控助手。它把租房合同审查、风险证据高亮、大白话解释、谈判改法、入住验房、退租证据包和城市补贴匹配整合到一个 React 工作台里，帮助用户在签字前看懂合同，在入住时留好证据，在退租时整理材料。

## 在线体验

- 线上作品：https://zu-xiao-shen.onrender.com
- GitHub 仓库：https://github.com/w3038177197/zu-xiao-shen
- 适用人群：毕业生、第一次租房的年轻人、租房弱势群体，以及陪家人看合同的人。

## 推荐演示路径

1. 打开线上作品，进入「租房审查」。
2. 载入「高风险整租合同」或粘贴自己的租房合同。
3. 点击「开始审查」，查看押金、涨租、维修、提前解除、入户检查、违约金和管辖条款风险。
4. 展开风险卡片，查看合同原文证据、大白话解释、谈判话术和替代条款。
5. 点击「全部采纳」，生成修订版合同草案和修改对比。
6. 下载租房解读报告或优化合同 DOCX，验证二次复查后的风险变化。
7. 打开「系统 AI 助手」，追问当前合同风险、退租证据准备或沟通话术。

## 项目亮点

- 租客视角：默认围绕押金、涨租、维修、违约、入户、管辖等真实租房痛点做严格审查。
- 证据绑定：每条风险都尽量关联合同原文，不凭空制造风险结论。
- 可行动输出：不仅提示“有风险”，还给出大白话解释、谈判话术、替代条款和 DOCX 报告。
- 全流程覆盖：从签约前合同审查，到入住验房留证，再到退租证据包和补贴匹配。
- 演示不断线：模型接口不可用时，会自动切换为本地租房风险规则和本地知识库兜底。

## 运行

```bash
npm install
npm run dev
```

默认访问 Vite 输出的本地地址。

如果要使用真实大模型接口，推荐同时启动后端代理：

```bash
copy .env.example .env
npm run dev:api
npm run dev
```

也可以一条命令同时启动前端和代理：

```bash
npm run dev:full
```

后端代理默认监听 `http://localhost:8787`，Vite 会把 `/api` 转发到代理服务。真实密钥放在后端环境变量里：

- `AI_PROXY_API_KEY`：通用兜底 API Key
- `DEEPSEEK_API_KEY`：DeepSeek
- `DASHSCOPE_API_KEY`：通义千问
- `ZHIPU_API_KEY`：智谱 GLM
- `MOONSHOT_API_KEY`：Moonshot
- `ARK_API_KEY`：火山方舟 / 豆包
- `HUNYUAN_API_KEY`：腾讯混元
- `AI_PROXY_BASE_URL` / `AI_PROXY_MODEL`：可选强制覆盖项，一般保持空，让前端服务商预设自动匹配

## 当前能力

- 首页总览：展示项目定位、四个核心模块入口、避坑流程教程和退租押金计算器原型。
- 补贴匹配：覆盖杭州、南京、武汉、成都、苏州、宁波、青岛、深圳、厦门、广州、北京、上海、天津、重庆、西安、长沙、合肥、郑州、济南、福州、无锡、佛山等城市，按所选城市只展示对应官方政策卡片。
- 租房审查：支持粘贴合同正文、载入多个演示合同模板，并可导入 TXT、MD、DOCX、PDF 和图片 OCR 合同，按租客视角识别押金、涨租、维修、解除、入户、管辖、格式条款等风险。
- 修改采纳：逐条或全部采纳风险修改建议，已采纳卡片会从待处理列表消失，并生成修订版合同草案和修改对比。
- 二次复查：采纳后的合同会直接写回编辑区，下载优化合同后再审查时可验证旧风险是否已经被改掉。
- 入住验房：按房屋类型和房间逐项记录状态，支持上传具体部位照片、标记瑕疵、生成针对部位的留证建议。
- 退租证据包：录入退租信息、勾选证据清单、生成沟通说明、整理行动清单和押金争议材料。
- 系统 AI 助手：通过“系统 AI 助手”打开，已接入当前模块、合同文本、风险结果、采纳记录、押金估算和本地知识库。
- AI 回复训练：内置风险审查、证据核验、退租沟通、补贴核验、回复排版和依据引用等技能，要求输出更专业、克制、少符号、可执行的回复。
- 知识库：本地规则与 `/api/rag/search` 轻量检索共同参与 AI 上下文，模型不可用时会切换到本地知识库兜底回复。
- DOCX 导出：租房解读报告、优化合同、入住验房报告、退租证据包均导出为 `.docx`，便于用户继续编辑和留档。
- 本地状态：审查历史、补贴匹配、验房记录、退租证据包和 AI 反馈使用浏览器本地存储保存。

## 已覆盖验证

- Playwright 端到端测试覆盖首页和避坑教程、系统 AI 兜底回复、租房审查与全部采纳、入住验房照片、DOCX 报告导出、补贴城市筛选。

- `npm run test:ai` 覆盖本地租房风险规则和 RAG 评估样本。
- `npm run lint` 与 `npm run build` 用于提交前基础质量检查。

## 后续扩展方向

- 把合同导入从纯文本抽取升级为结构化解析，自动识别甲乙方、租金、押金、租期、地址和关键条款位置。
- 把本地 RAG 升级为正式向量数据库，例如 pgvector、Qdrant、Milvus 或 Elasticsearch，并加入法规和政策版本更新任务。
- 给后端代理增加登录鉴权、限流、调用审计、日志脱敏和用户额度控制。
- 扩展入住验房照片的 AI 识别能力，自动识别墙面、地板、门窗、水电燃气和家具家电瑕疵，并把照片索引写进 Word 报告。
- 增加房源行情参考、房东/中介信用提示、维权路径和投诉材料生成。
- 把押金计算器从首页原型升级为完整退租结算工具，支持费用凭证、正常损耗判断和争议金额拆解。
